# Jolt 刚体物理核心算法 — 概述与架构分层

## 一、本系列文档目标

通过拆解 Jolt Physics 的源码，提取出**纯粹的刚体物理数学算法**，
剥离工业化包装（线程安全、对象管理、序列化），为后续仿写一个
**单文件、简单刚体物理引擎** 提供完整的知识参考。

## 二、三层架构拆解

| 层级 | 代表文件/目录 | 性质 | 仿写需要？ |
|------|-------------|------|-----------|
| **外壳层** | `BodyInterface`, `BodyManager`, `Factory`, `RTTI`, `Stream`, `CollisionGroup`, `ObjectLayer` | API包装、线程安全锁、序列化、对象池管理 | **不需要** |
| **调度层** | `PhysicsSystem.cpp`, `IslandBuilder`, `LargeIslandSplitter`, `JobSystem`, `BroadPhase/QuadTree` | 多线程Job图、Island并行分割、QuadTree宽相位 | **大幅简化**（单线程 + O(n²) 宽相位） |
| **核心算法层** | `Geometry/GJK*.h`, `EPA*.h`, `ClosestPoint.h`, `AxisConstraintPart.h`, `MotionProperties.inl`, `ManifoldBetweenTwoFaces.cpp`, `ClipPoly.h` | 纯数学算法 | **必须实现** |

## 三、核心算法文件地图

所有路径相对于 `build/_deps/joltphysics-src/Jolt/`：

### 数据定义
| 文件 | 作用 |
|------|------|
| `Physics/Body/Body.h` | 刚体结构：位置、旋转、摩擦、弹性、形状引用 |
| `Physics/Body/MotionProperties.h` | 运动状态：线/角速度、逆质量、逆惯性张量、力/扭矩累积、阻尼 |
| `Physics/Body/MotionProperties.inl` | 速度积分、阻尼、陀螺力的具体实现 |
| `Physics/Body/Body.inl` | 位置/旋转积分（四元数更新）、impulse施加 |
| `Physics/Body/MassProperties.h` | 质量+惯性张量 → 对角化分解（仅创建时用） |

### 碰撞检测
| 文件 | 作用 |
|------|------|
| `Geometry/ConvexSupport.h` | Support函数接口：MinkowskiDifference、ConvexRadius膨胀 |
| `Geometry/ClosestPoint.h` | 单纯形最近点：线段/三角形/四面体上到原点最近点（GJK子程序） |
| `Geometry/GJKClosestPoint.h` | GJK算法：两凸体距离计算，~200行核心循环 |
| `Geometry/EPAPenetrationDepth.h` | EPA算法：穿透深度与法线计算 |
| `Geometry/EPAConvexHullBuilder.h` | EPA所用凸包数据结构（三角形面+优先队列） |
| `Geometry/ClipPoly.h` | Sutherland-Hodgman多边形裁剪（接触面生成） |
| `Physics/Collision/ManifoldBetweenTwoFaces.cpp` | 接触流形生成：从GJK/EPA结果生成多个接触点 |

### 约束求解
| 文件 | 作用 |
|------|------|
| `Physics/Constraints/ConstraintPart/AxisConstraintPart.h` | 1-DOF约束求解器（最核心！碰撞法线+摩擦都用这个） |
| `Physics/Constraints/ConstraintPart/SpringPart.h` | 弹簧/软约束的bias与softness计算 |
| `Physics/Constraints/ContactConstraintManager.cpp` | 接触约束的组装与求解调度 |

### 主循环
| 文件 | 作用 |
|------|------|
| `Physics/PhysicsSystem.cpp` | 完整物理更新循环（Job依赖图） |
| `Physics/PhysicsSettings.h` | 所有可配置常量（迭代次数、Baumgarte系数等） |

## 四、完整物理更新流水线（单步简化顺序）

```
Step 0: 前处理 — 清零 force/torque，调用 StepListener
Step 1: 速度积分 — 重力+外力→速度，阻尼，陀螺力（可选）
Step 2: 宽相位碰撞检测 — AABB重叠对收集
Step 3: 窄相位碰撞检测 — GJK求距离，EPA求穿透
Step 4: 接触流形生成 — ClipPoly裁剪，PruneContactPoints精简到≤4点
Step 5: 约束组装 — 每个接触点创建 法线约束+2个切线摩擦约束
Step 6: Warm Start — 用上帧λ乘(dt_new/dt_old)缩放后施加
Step 7: 速度约束求解 — 迭代10次(默认)，每次先摩擦后法线
Step 8: 位置积分 — p += v*dt, q = Quat(ω*dt) * q
Step 9: 位置约束修正 — Baumgarte(β=0.2)，迭代2次(默认)
Step 10: 休眠检测与回调
```

> 后续文档 01~09 将逐一展开每个步骤的数学细节与实现要点。
